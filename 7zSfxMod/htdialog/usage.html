Usage:

HTDialog.exe [options] <filename> [arguments]

Available options:

  • /hidden
    Hides the associated dialog window
  • /zoom:<percentage>
    Scales the associated dialog window accordingly
  • /features:"<semicolon-delimited list of values>"
    Controls various aspects of the associated dialog window
    Possible values:
    • dialogLeft: <number> { cm | mm | in | pt | pc | em | ex | px }
    • dialogTop: <number> { cm | mm | in | pt | pc | em | ex | px }
    • dialogWidth: <number> { cm | mm | in | pt | pc | em | ex | px }
    • dialogHeight: <number> { cm | mm | in | pt | pc | em | ex | px }
    • center: { yes | no | 1 | 0 | on | off }
    • edge: { sunken | raised }
    • resizable: { yes | no | 1 | 0 | on | off }
    • scroll: { yes | no | 1 | 0 | on | off }
    • status: { yes | no | 1 | 0 | on | off }
    • unadorned: { yes | no | 1 | 0 | on | off }
